# 本期功能说明 (v2.0)

## 一、版本概述

v2.0 是智多星智能客服系统的首个完整版本，覆盖从用户接入到智能应答、再到转人工的完整闭环。

核心目标：**实现 80% 常见问题的自动应答，人工转接率控制在 20% 以内。**

## 二、新增功能

### 2.1 智能路由

基于 BERT 微调的意图识别模型，支持 50+ 业务意图分类。系统根据意图置信度自动决定走直接回答、多轮澄清还是转人工。

### 2.2 实时思考流

采用 Server-Sent Events (SSE) 协议，将 Agent 的推理步骤（意图识别 → 知识检索 → 答案生成）以流式推送至前端，让用户实时看到 AI "正在思考什么"。

### 2.3 混合知识检索

- 向量检索：基于 text-embedding-3-large 的语义召回
- 关键词检索：BM25 精确匹配
- 知识图谱：Neo4j 实体关系查询
- 重排序：Cross-encoder Reranker 精排

### 2.4 智能转人工

检测用户情绪（基于情感分析模型）和问题复杂度，在最佳时机自动触发转人工。转接时附带 AI 生成的对话摘要，减少坐席理解成本。

### 2.5 自进化引擎

E→E→E→R 循环：执行(Execute) → 评估(Evaluate) → 提取(Extract) → 回注(Retrieve)。对话数据自动评估质量，高质量案例提取为知识条目回注到知识库。

## 三、性能指标

| 指标 | 目标值 | 当前状态 |
|:---|:---|:---|
| 首次响应时间 | < 2s | 1.3s |
| 意图识别准确率 | > 90% | 92.5% |
| 自动应答率 | > 80% | 78% |
| 用户满意度 | > 4.2/5 | 4.5/5 |

## 四、已知限制

1. 当前不支持多语言，仅限中文
2. 语音输入暂未接入，仅支持文本
3. 批量操作（如批量退款）需走人工通道
